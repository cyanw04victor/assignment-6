html

HTML (HyperText Markup Language) is the foundation of every web page. It is not a programming language; it is a markup language used to describe the structure and meaning of content on the web.

Main Uses of HTML

Page structure

Defines headings, paragraphs, sections, and layout blocks.

Common tags: html, head, body, header, nav, main, section, article, footer

Links and navigation

Creates connections between pages and websites.

Uses <a href="...">link text</a>

Supports internal bookmarks with href="#section-id"

Media embedding

Common tags: img, audio, video, iframe

Inserts pictures, audio, and video into a page.

Makes pages richer and more interactive

Forms and user input

Collects information from visitors.

Uses <form>, <input>, <textarea>, <select>, <button>

Essential for contact forms, sign-ups, search, and surveys

Tables and data display

Organizes information in a structured format.

Uses <table>, <tr>, <td>, <th>

Good for schedules, reports, and comparison data

Shows structured data in rows and columns.

Accessibility and semantics

Ensures content is accessible to users with disabilities.

Uses semantic elements to convey meaning and structure.

Semantic tags improve search engine ranking and help screen readers.

Provides meaning to content so browsers and assistive tools understand it.
